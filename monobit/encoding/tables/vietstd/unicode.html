=============================================
	    VISCII 1.1 to Unicode 1.1 (ISO 10646) MAPPING
	    =============================================

	    
This document contains two segments:

    1) Code fragment of a table mapping the 256 VISCII 1.1
       character codes to Unicode 1.1.  This is the same
       as the table found in AT&T's Plan 9 TCS character set
       conversion utility, available for anonymous FTP from
       research.att.com:dist/tcs.tar.Z.

    2) Line-by-line mapping of the 134 Vietnamese-specific
       letters, from VISCII 1.1 to Unicode 1.1, including
       their VIQR mnemonic representation and the English
       character names.

For further information, consult the VISCII documentation, or
contact Viet-Std@Haydn.Stanford.EDU.

===================
