==Konatu Description==

===How to install(for BeOS and zeta)===
It installs in /boot/home/config/fonts/ttfonts/

===Attention===

Be Bitmap Font(4bit and 8 gradate bitmap font)

in this archive
(Konatu_8 ......Konatu_20 ,and KonatuTohaba_8 ...... KonatuTohaba_20)

These can be used only with BeOS/Zeta.
These can't be used with HAIKU. 

===About this version===

* [ Be Bitmap Font ] Grif in the Chinese character part was corrected. 
* [True Type Font] Embedded 1bit Bitmap font.But only 12pixel and 13 pixel size.
